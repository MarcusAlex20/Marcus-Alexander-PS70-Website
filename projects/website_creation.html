Final Project Proposal

Project Vision

The final project that I will be undergoing for PS70 is creating a smart headgear that can be used to monitor combat sports athletes head impacts. The motivation behind this is while there has been innovation in many sports, boxing and MMA have been largely left untoched.

Project Description & Technical Components

The smart headgear will incorporate various sensors to monitor and analyze head impacts during training and competition:

  • Impact force measurement and analysis
  • Real-time monitoring of head movement and acceleration
  • Data collection for long-term health assessment
  • Integration with mobile applications for immediate feedback
  • Accelerometer and gyroscope sensors for impact detection
  • Microcontroller for data processing
  • Wireless communication module for real-time data transmission
  • Battery management system for extended use
