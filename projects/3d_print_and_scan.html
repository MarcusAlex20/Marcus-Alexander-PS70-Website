3D Print & Scan

PS70 Project 05 3D Printing Scanning

Project Overview

This project involved 3D printing and scanning techniques, combining digital design with physical fabrication. While initially I intended to make a giraffe for this project, as I learned more I acquired more skills and re-did this assignment. For the remake, I created a bust of Antonius Pius from The Louvre, demonstrating advanced 3D printing capabilities and attention to detail.

3D Printing

Design

Created a 3D model of a giraffe using Fusion 360.

Preparation

Prepared the model for printing using slicing software.

Printing

Executed the 3D printing process and monitored the build.

3D Scanning

For the scanning portion of the project, I scanned a mannequin in the PS70 lab using Poly.cam. This allowed me to create a detailed 3D model of the mannequin that can be viewed and interacted with directly in the browser.

Challenges & Solutions

Printing Challenges

Faced challenges in maintaining detail and surface quality of the bust.

Solutions:

  • Optimized layer height for better detail preservation
  • Implemented strategic support structures
  • Refined post-processing techniques for surface finish
  • Adjusted print speed for optimal quality
