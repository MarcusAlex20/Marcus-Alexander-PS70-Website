Smart Sleep Sensor

This week Callum and I made a smart sleep sensor. The way that this sensor connected to our HTML website that we made was via Bluetooth.

The sensor measures the following parameters:

  • Temperature (°C): The optimal range is between 16°C and 20°C. Maintaining this range ensures a comfortable and restful sleep environment.
  • Humidity (%): The optimal range is between 30% and 50%. Proper humidity levels help prevent dryness and promote better air quality for sleeping.

Sensor Used: Nordic Thingy:53

Photo of the Sensor:

Code Environment Setup

Below is the configuration used for setting up the Nordic Thingy:53:

CONFIG_BT=y
CONFIG_BT_PERIPHERAL=y
CONFIG_BT_DEVICE_NAME="Thingy53"
CONFIG_SENSOR=y
CONFIG_PWM=y
CONFIG_BME680=y
CONFIG_IPC_SERVICE=y
CONFIG_LED_PWM=y
CONFIG_BOARD_ENABLE_CPUNET=y

Rationale: The board was confirgured using Zephyr OS, a platform to program the nRF53 MCU. This was Callum's area of expertise so he took the lead on it. This allowed us to leverage Bluetooth configurations, sensor functionalities, and controlling PWM for LEDs.

Sensor Integration and Data Acquisition

sensor_sample_fetch(bme);
sensor_channel_get(bme, SENSOR_CHAN_AMBIENT_TEMP, &temperature);
sensor_channel_get(bme, SENSOR_CHAN_HUMIDITY, &humidity);
sensor_channel_get(bme, SENSOR_CHAN_PRESS, &pressure);
