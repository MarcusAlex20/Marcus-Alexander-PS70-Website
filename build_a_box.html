Assignment 02: Build a Box

Building the Box In Fusion 360

ITERATION 1: Building my finger joint box was very straightforward and allowed me to get a better understanding of how to use CAD Fusion 360. Lessons that I learned were how to use simple shortcuts such as shift + select to grab multiple objects and constrain them to equal. I also became proficient with adding and changing parameters. Additionally, I became very familiar with using construction lines to mirror objects. I became very familiar with the extrude tool and I am excited to see how I can use it later on in the semester. I used the display component tool colors to ensure that the finger joints fit flush. At this moment, I have not laser cut my box yet; I will give an additional update once I have done so.

These images show the box I built during week 2 and the parameters used for its construction. Ensuring accuracy in these parameters was crucial for the success of the project. It was a challenging but rewarding experience.

Challenges & Solutions

Metric System Understanding

Initially created a miniature box due to misunderstanding metric measurements.

Solution

Learned to properly interpret metric measurements and adjusted parameters accordingly. Turned the miniature box into an opportunity for innovation by using it as a screw storage compartment.

Fusion 360 Learning Curve

Navigating the complex interface and tools of Fusion 360 was challenging.

Solution

Learned keyboard shortcuts (shift + select), mastered parameter management, and became proficient with construction lines and the extrude tool through hands-on practice.

Future Improvements

  • Add decorative elements to the box design
  • Experiment with different joint types
  • Create a more complex box with multiple compartments
  • Explore different materials for the box construction
