Spring 2024 PS70

Building the Box In Fusion 360

Box parameters in Fusion 360

3D model of the box design

The box was designed using Fusion 360, with careful consideration given to dimensions and assembly requirements. The design process involved creating a parametric model that could be easily modified and iterated upon.

Final Result

The completed box

The final box was successfully assembled and tested. A miniature version was also created to demonstrate the scalability of the design.

Technical Details


// Fusion 360 Parameters
box_width = 100mm
box_height = 80mm
box_depth = 60mm
material_thickness = 3mm
finger_joint_width = 10mm
finger_joint_count = 5

Challenges & Solutions

Metric System Understanding

Initially created a miniature box due to misunderstanding metric measurements.

Solution

Learned to properly interpret metric measurements and adjusted parameters accordingly. Turned the miniature box into an opportunity for innovation by using it as a screw storage compartment.

Fusion 360 Learning Curve

Navigating the complex interface and tools of Fusion 360 was challenging.

Solution

Learned keyboard shortcuts (shift + select), mastered parameter management, and became proficient with construction lines and the extrude tool through hands-on practice.

Future Improvements

  • Add decorative elements to the box design
  • Experiment with different joint types
  • Create a more complex box with multiple compartments
  • Explore different materials for the box construction
