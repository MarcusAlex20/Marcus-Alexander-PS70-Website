05: 3D Print & Scan

Spring 2024 PS70

Project Overview

This week I attempted to 3D print a giraffe on Fusion. Unfortunately, I had a 3D print error so I sadly do not have the overall object, but I will print it later this week.

Design Process

Step 1: 3D Modeling

I created a detailed giraffe model in Fusion 360, focusing on capturing the distinctive features of a giraffe including its long neck, spots, and proportions.

Step 2: Slicing

I prepared the model for printing by slicing it in PrusaSlicer, adjusting settings for optimal print quality and support structures.

Step 3: Printing Attempt

I attempted to print the model but encountered an error during the printing process. The files are saved for a future printing attempt.

Technical Details


// 3D Print Specifications
printer = "Prusa i3 MK3S+"
material = "PLA"
layer_height = "0.2mm"
infill = "20%"
supports = "Tree supports"
estimated_print_time = "4 hours"

Challenges & Solutions

Print Failure

The 3D print encountered an error during the printing process.

Solution

I've saved all the necessary files and will attempt to print again later this week. I'll also review the printer settings to identify potential issues.

Model Complexity

Creating a detailed giraffe model with proper proportions was challenging.

Solution

I used reference images and focused on key features to create a recognizable giraffe silhouette while maintaining printability.

Future Improvements

  • Successfully complete the 3D print
  • Add more detail to the giraffe's spots and texture
  • Experiment with different materials and colors
  • Create a family of African animals
